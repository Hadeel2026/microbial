This study uses two publicly accessible datasets from the DeepBacs repository to develop a detailed workflow to improve resolution and segment bacterial microscopy images. The first dataset (Zenodo: 5550933)[\url{https://zenodo.org/records/5550933}] consists of WF microscopy images of Staphylococcus aureus, paired with expert-defined segmentation masks for both bright-field and fluorescence imaging. This dataset provides a foundation for training and validating segmentation models to accurately identify bacterial cells. The second dataset (Zenodo: 5551141)[\url{https://zenodo.org/records/5551141}] provides paired WF and SIM images of the same bacterial species, which aids supervised learning for super-resolution reconstruction tasks.
